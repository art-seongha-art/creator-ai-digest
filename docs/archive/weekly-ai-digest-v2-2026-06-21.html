언리얼 엔진 5.8, 에디터에 MCP 서버 탑재 — AI 에이전트가 자연어로 씬을 조작한다

Epic Games가 6월에 공개한 언리얼 엔진 5.8에는 에디터 안에서 Model Context Protocol(MCP) 서버를 실행하는 실험적(experimental) Unreal MCP 플러그인이 포함됩니다. 플러그인을 켜면 Claude Code, Codex, Gemini 같은 MCP 호환 AI 에이전트가 기본 로컬 MCP 엔드포인트로 에디터에 붙어 액터 배치, 조명·머티리얼 설정, 블루프린트 탐색, 자동화 테스트 같은 작업을 '도구'처럼 호출할 수 있습니다. 이 항목은 한 창작자의 Threads 공유에서 출발했지만, 공식 Epic Developer 문서에 'Unreal MCP in Unreal Editor'로 기재된 사실로 교차 확인했으며 문서에는 아직 실험 단계임이 명시되어 있습니다.

인사이트: 실감미디어, 가상 프로덕션, 전시 프리비즈를 다루는 창작자에게 이 변화는 '엔진 조작을 자연어로 지시하는' 제작 워크플로우의 출발점입니다. 도시·공간 시뮬레이션 셋업이나 라이팅 시안처럼 반복적인 작업을 에이전트에게 맡기고, 작가는 연출과 서사 판단에 집중하는 분업을 실험해 볼 수 있습니다. 다만 공식 문서가 명시하듯 experimental 단계이고 로컬 서버를 여는 구조이므로, 결과 검수·버전 호환·보안 설정을 반드시 함께 확인해야 하며 'AI가 알아서 게임을 만든다'는 식의 과장과는 거리가 있습니다.
