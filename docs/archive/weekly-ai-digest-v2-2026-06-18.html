LTX Trainer 공개 — 비디오 생성도 직접 LoRA·IC-LoRA 학습 단계로

6월 18일자 핵심 지정 항목입니다. Lightricks가 LTX-2/LTX-2.3용 오픈소스 학습 프레임워크를 공개하면서, 사용자가 직접 데이터셋을 준비해 영상 생성 모델을 LoRA, full fine-tuning, IC-LoRA 방식으로 커스터마이징할 수 있는 길이 열렸습니다.

인사이트: 새 영상 모델 하나가 더 나온 정도가 아니라, 창작자가 프롬프트 소비자에서 모델 커스터마이징 제작자로 넘어가는 흐름입니다. 작품 스타일, 캐릭터 일관성, 움직임 레퍼런스, 립싱크·복원·후반작업 자동화 교육에 연결됩니다.
