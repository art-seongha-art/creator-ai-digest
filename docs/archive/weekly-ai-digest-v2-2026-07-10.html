Google 6월 AI 업데이트: Gemini 이미지·비디오 워크플로우가 더 빠르고 저렴한 방향으로

Google은 6월 AI 업데이트에서 더 빠르고 비용 효율적인 Gemini Image 모델과 Gemini Omni Flash API 공개 프리뷰를 강조했습니다. 텍스트로 3~10초 영상을 만들고, 정지 이미지를 애니메이션화한 뒤 대화식으로 수정하는 흐름이 핵심입니다.

인사이트: 수업에서는 ‘스토리보드 한 장→3초 영상→대화형 수정’ 과제를 만들기 좋고, 디자인팀은 짧은 콘셉트 검증 영상을 빠르게 뽑는 용도로 살펴볼 수 있습니다.

arXiv cs.AI2026.07.09

OpenCoF: 영상 생성 과정을 ‘프레임으로 이어지는 추론’으로 보는 연구

OpenCoF 논문은 영상 생성 모델이 시간적으로 연결된 프레임을 만들며 추론 과정을 드러낼 수 있다는 관점에서 출발합니다. 텍스트 체인오브쏘트와 달리, 장면 변화 자체가 문제 해결의 중간 단계가 될 수 있다는 점을 실험합니다.

인사이트: 월드모델·인터랙티브 콘텐츠를 가르칠 때 ‘생성된 장면의 다음 프레임이 어떤 판단을 보여주는가’를 토론하는 자료로 쓸 수 있습니다. 아직 연구 단계라 제작 도구로 바로 쓰기보다는 개념 이해용으로 보는 것이 좋습니다.

Meta AI2026.07.08

Meta, Muse Image와 Muse Video 소개: 레퍼런스 기반 편집과 네이티브 오디오 영상에 초점

Meta가 Muse Image와 Muse Video를 소개했습니다. Muse Image는 지시문 준수, 정밀 편집, 여러 레퍼런스 조합을 강조하고, Muse Video는 시각 품질과 시간적 일관성, 네이티브 오디오 지원을 앞세웁니다.

인사이트: 브랜드 캠페인이나 캐릭터 세계관을 다루는 팀은 ‘여러 참고 이미지로 톤을 고정하고 짧은 영상으로 확장’하는 제작 실험을 검토할 만합니다. 공식 공개 범위와 사용 조건은 계속 확인이 필요합니다.

Runway2026.07.08

Runway Dev 공개: 이미지·영상·오디오·실시간 캐릭터 모델을 하나의 API로 묶다

Runway가 개발자용 AI 미디어 플랫폼 Runway Dev를 공개했습니다. 한 API에서 이미지, 영상, 오디오, 실시간 캐릭터 모델을 연결할 수 있어 웹 서비스나 교육용 제작 도구 안에 생성형 미디어 기능을 직접 넣기 쉬워집니다.

인사이트: 디자이너와 교육자는 ‘프롬프트→이미지→짧은 영상→음향’ 같은 수업용 프로토타입을 별도 툴 이동 없이 앱 안 워크플로우로 설계해볼 수 있습니다.

Meta AI2026.07.08

SAM Audio: 텍스트·영상 단서로 원하는 소리만 분리하는 오디오 모델

Meta는 SAM Audio를 ‘오디오용 Segment Anything’에 가까운 통합 모델로 소개했습니다. 텍스트 설명, 영상 단서, 시간 구간 표시를 이용해 복잡한 사운드에서 특정 소리만 분리하는 것이 목표입니다.

인사이트: 영상 수업이나 사운드 디자인 작업에서 현장 녹음의 발소리, 대사, 배경음을 분리해 재편집하는 실습에 바로 연결됩니다. 실제 사용 가능 범위와 라이선스는 공식 페이지를 확인해야 합니다.
